# 💪 Gym Tracker

A full-stack workout tracking web app built using:

- HTML, CSS, JavaScript  
- PHP  
- MySQL  

---

## 🚀 Features

- 🔐 User authentication (Login/Register)
- 🏋️ Start and log workouts
- 📊 Dashboard with stats
- 🔍 Exercise search functionality
- 📁 Workout history tracking
- 🎯 Clean dark UI (ChatGPT-inspired)
